# AI Email Automation System

## 🚀 What this project does
This project automates email replies using n8n workflows and AI.

## ⚙️ Tech Used
- n8n
- OpenAI API
- Webhooks

## 🔄 Workflow
- Receives email
- Classifies message
- Generates reply using AI
- Sends response automatically
